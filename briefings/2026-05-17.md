**Impact:** For teams weighing open-model deployment against closed API calls, this is a meaningful capability shift. The DeepSeek V4 1M-context efficiency story is particularly relevant for telecom/OSS contexts where operational logs, device configs, or fault correlation data can be large: long-context is now viable without proportional cost blowout. Gemma 4's 26B MoE offers an on-premise or edge option where active parameter count drives GPU memory and cost. The breadth of Apache-2.0 and similarly permissive releases materially reduces licensing friction for production use in regulated or on-premise environments.
